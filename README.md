🚀 Velozity Project Tracker UI

A modern, high-performance Project Management Frontend Application built as part of a technical hiring assessment. This application demonstrates advanced frontend engineering concepts including multi-view data synchronization, custom drag-and-drop, virtual scrolling, and simulated real-time collaboration.

📌 Overview

The Velozity Project Tracker UI is designed to manage and visualize tasks efficiently across multiple views. It ensures smooth performance even with large datasets while maintaining a clean and responsive user interface.

The application focuses on:

Performance optimization
Scalable UI architecture
Seamless user interactions
Real-time collaboration simulation
✨ Features
🔄 Multi-View Support
Board View (Kanban-style task management)
List View (Structured task listing)
Timeline View (Time-based visualization)
All views are synchronized with the same dataset
🎯 Custom Drag-and-Drop
Built from scratch (no external libraries)
Smooth task movement across columns
Real-time UI updates
⚡ Virtual Scrolling
Efficient rendering of large datasets
Improves performance and reduces DOM load
👥 Live Collaboration Indicators
Simulates active users
Displays presence on tasks
Enhances team awareness
🎨 Modern UI/UX
Responsive layout
Clean design with cards and shadows
Hover effects and smooth animations
🛠️ Tech Stack
HTML5 – Structure
CSS3 – Styling (Flexbox, Grid, Animations)
JavaScript (ES6+) – Logic & Interactions
